Instructions for how to use the prebuilt images for https://android-build.linaro.org/builds/~linaro-android/panda-jb-gcc47-tilt-tracking-blob/

Validated on Ubuntu 12.04 LTS using build 16 in this series

1. Scroll down to the 'Downloads' section.

Click on each link to download:
     target/product/pandaboard/system.tar.bz2
     target/product/pandaboard/boot.tar.bz2
     target/product/pandaboard/userdata.tar.bz2


2. Grab linaro-image-tools

Run these commands to get all the dependencies for linaro-image-tools and the tip of linaro-image-tools

sudo add-apt-repository ppa:linaro-maintainers/tools
sudo apt-get update
sudo apt-get install linaro-image-tools

If you're using a "tip" build do this, if you're using a released build (with a -release or from releases.linaro.org) then don't

sudo apt-get install bzr
bzr branch lp:linaro-image-tools


3. Program an SD card

Save automount settings (Gnome)

TMP1=$(dconf read /org/gnome/desktop/media-handling/automount)
TMP2=$(dconf read /org/gnome/desktop/media-handling/automount-open)

Disable automount (Gnome)

dconf write /org/gnome/desktop/media-handling/automount false
dconf write /org/gnome/desktop/media-handling/automount-open false

Insert an SD card

Run 'dmesg' and look for a line that looks like the following at the end of the log

[288582.790722]  sdc: sdc1 sdc2 sdc3 sdc4 < sdc5 sdc6 >

WARNING! In the next step, make sure you use /dev/"whatever you see above". You can erase your hard drive with the wrong parameter.

Run the tip of linaro image tools

./linaro-image-tools/linaro-android-media-create --mmc /dev/sdc --dev panda --system system.tar.bz2 --userdata userdata.tar.bz2 --boot boot.tar.bz2

Restore automount (Gnome)

dconf write /org/gnome/desktop/media-handling/automount $TMP1
dconf write /org/gnome/desktop/media-handling/automount-open $TMP2

Remove the SD card from the programmer and plug it into the Panda.


5. Look at the console output

Plug in a USB-to-serial converter and run this on the command line:

minicom -D /dev/ttyUSB0 -w -C minicom.txt
